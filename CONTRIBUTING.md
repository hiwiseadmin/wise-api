# Contributing

Thank you for considering contributing to this package!

## Development Setup

```bash
git clone https://github.com/wiseadmin/wise-api.git
cd wise-api
composer install
```

## Coding Standards

- PSR-12 coding style
- PSR-4 autoloading
- PHP 8.1+ features preferred

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Write tests for new functionality
4. Ensure all tests pass
5. Submit a pull request with clear description

## Running Tests

```bash
./vendor/bin/phpunit
```

## Security

If you discover any security related issues, please email security@wiseadmin.dev instead of using the issue tracker.
